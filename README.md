# 🏓 Pong Game

A classic **Pong arcade game** built with **Python Turtle Graphics** and **Object-Oriented Programming (OOP)**. Two players control paddles to bounce the ball, score points, and compete in a fast-paced retro game.

## ✨ Features

* 🎮 Two-player paddle controls
* ⚽ Ball movement and bounce physics
* 🧱 Wall collision detection
* 🏓 Paddle collision detection
* 🏆 Real-time score tracking
* ⚡ Increasing ball speed during gameplay
* 🎨 Retro arcade-style interface

## 🛠️ Built With

* Python 3
* Turtle Graphics
* Object-Oriented Programming (OOP)
* Event Handling
* Collision Detection
